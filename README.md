# TechPassionateHx Portfolio 🚀

Welcome to my personal portfolio website! This project is designed to showcase my works, UI/UX designs, and technical experience in a clean, minimalist layout.

## ✨ Features

* **Responsive Grid:** Fully optimized for mobile, tablet, and desktop screens.
